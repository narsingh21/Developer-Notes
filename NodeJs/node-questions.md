## what is Node ?
- It an enviroment which can run Node js outside of browser?

## Diffrence between running js in browser v/s in Node?
- When JS code is running in browser. It has the access of  Browser apis like DOM,Audio-video's api, storgare api, geolocation api and so on. These apis won't be availabel in Node enviroment.
- When JS code is running in Node. It has filesystem access (read,write file ).
